# Word Count Tool

Word Count Tool is a **web-based application** that analyzes text input or uploaded `.txt` files to provide **word count, character count, line count, and word frequency analysis** in real time.

Built with **Next.js**, it offers a fast, responsive interface for text analysis tasks.

---

## Features

* Live word count
* Character count
* Line count
* Word frequency analysis
* `.txt` file upload support
* Real-time updates as you type
* Clean and responsive UI

---

## Installation

Make sure you have the following installed:

* **Git**
* **Node.js (LTS version recommended)**

Clone the repository and install dependencies:

```bash
git clone https://github.com/SarthakhostZ/Word-Count-Tool.git
cd Word-Count-Tool
npm install
```

---

## Usage

Start the development server:

```bash
npm run dev
```

Expected output:

```text
▲ Next.js 15.x.x
- Local:        http://localhost:3000
- Ready in Xs
```

Open your browser and visit:

```text
http://localhost:3000
```

You can now:

* Paste or type text directly into the editor
* Upload a `.txt` file for analysis
* View word, character, line counts, and word frequency instantly

## Troubleshooting

If something doesn’t work:

1. Check the terminal for error messages
2. Ensure Node.js is installed correctly:

   ```bash
   node -v
   ```
3. Reinstall dependencies:

   ```bash
   rm -rf node_modules package-lock.json
   npm install
   ```
4. Restart the development server

If issues persist, open an issue on GitHub.
